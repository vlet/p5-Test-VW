# NAME

Test::VW - always green cpantesters

# SYNOPSIS

    use Test::More;
    use Test::VW;

# DESCRIPTION

Test::VW detects when your test are being run on cpantesters (checks for
AUTOMATED\_TESTING environment variable) and makes them always pass.
